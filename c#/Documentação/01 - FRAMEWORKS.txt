## ASP.NET CORE
# FRAMEWORK PRINCIPAL DA APLICAÇÃO WEB
# RESPONSÁVEL POR:
# - MVC (MODELS, VIEWS, CONTROLLERS)
# - PIPELINE HTTP
# - INJEÇÃO DE DEPENDÊNCIA
# - CONFIGURAÇÃO GERAL DA APLICAÇÃO


## ENTITY FRAMEWORK CORE (EF CORE)
# ORM (OBJECT RELATIONAL MAPPER)
# RESPONSÁVEL POR:
# - MAPEAR CLASSES C# PARA TABELAS DO BANCO DE DADOS
# - GERAR E ATUALIZAR O BANCO VIA MIGRATIONS
# - ABSTRAIR O ACESSO AO BANCO
#
# EXEMPLO:
# A CLASSE Usuario.cs ORIGINA A TABELA Usuarios


## ASP.NET CORE IDENTITY
# FRAMEWORK DE AUTENTICAÇÃO E AUTORIZAÇÃO
# RESPONSÁVEL POR:
# - LOGIN / LOGOUT
# - GERENCIAMENTO DE USUÁRIOS
# - SENHAS (HASH)
# - ROLES (PERFIS)
# - CLAIMS
#
# UTILIZA EF CORE INTERNAMENTE PARA:
# - CRIAR TABELAS DE USUÁRIOS
# - GERENCIAR DADOS DE AUTENTICAÇÃO


## REGRA DE OURO
# EF CORE = BANCO DE DADOS
# IDENTITY = SEGURANÇA / USUÁRIOS
# ASP.NET CORE = ESTRUTURA DA APLICAÇ
